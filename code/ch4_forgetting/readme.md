# Chapter 4: Forgetting Trajectories

| RQ | Title | Research Question |
|-----|-------|-------------------|
| 4.1.1 | Functional Form of Forgetting Trajectories | Which functional form best describes episodic forgetting trajectories across a 6-day retention interval? |
| 4.1.2 | Evidence for Two-Phase Forgetting (Rapid then Slow) | Do data support a two-phase model with rapid initial decline (Day 0-1) followed by slower decay (Day 1-6)? |
| 4.1.3 | Does age affect baseline memory ability or forgetting rate? | Do older adults show lower baseline episodic memory (intercept) and/or faster forgetting (steeper slope) compared to younger adults? |
| 4.1.4 | Between-Person Variance in Forgetting Rates | What proportion of variance in forgetting rate (slopes) is between-person (stable individual differences) vs within-person (measurement error)? |
| 4.1.5 | Individual Clustering | Can participants be grouped into latent classes based on their forgetting trajectories (intercepts and slopes)? |
| 4.2.1 | Domain-Specific Forgetting Trajectories (What/Where/When) | Are there domain-specific differences in the rate and pattern of episodic forgetting over 6 days? |
| 4.2.2 | Differential Consolidation Across Memory Domains | Do memory domains (What/Where) show different rates of forgetting during the early consolidation window (Day 0->1) versus later decay (Day 1->6)? |
| 4.2.3 | Domain-Specific Age Effects on Forgetting | Does the effect of age on forgetting rate vary by memory domain (What, Where)? |
| 4.2.4 | IRT-CTT Convergent Validity | Do IRT theta scores and CTT mean scores yield the same conclusions about domain-specific forgetting trajectories? |
| 4.2.5 | Does purified IRT item set change CTT conclusions? | If we compute CTT scores using only IRT-retained items (post-purification), do conclusions differ from full-item CTT? |
| 4.2.6 | Domain-Specific Variance Decomposition | What proportion of variance in forgetting rate is between-person versus within-person for each memory domain (What, Where)? |
| 4.2.7 | Domain-Based Clustering | Can participants be grouped into latent classes based on domain-specific forgetting trajectories (What/Where intercepts and slopes)? |
| 4.3.1 | Do Free Recall, Cued Recall, and Recognition Exhibit Different Forgetting Trajectories? | Are there paradigm-specific differences in the rate and pattern of episodic forgetting over 6 days? |
| 4.3.2 | Linear Trend in Forgetting Rate Across Paradigms | Does forgetting rate decrease monotonically from Free Recall -> Cued Recall -> Recognition, consistent with an ordered retrieval support gradient? |
| 4.3.3 | Paradigm Consolidation Window | Do retrieval paradigms (Free Recall, Cued Recall, Recognition) show different consolidation benefits during the early consolidation window (Day 0->1) versus later decay period (Day 1->6)? |
| 4.3.4 | Age × Paradigm Interactions | Does the effect of age on forgetting rate vary by retrieval paradigm? |
| 4.3.5 | IRT-CTT Convergence for Paradigm-Specific Forgetting | Do IRT theta scores and CTT mean scores yield the same conclusions about paradigm-specific forgetting trajectories for Free Recall, Cued Recall, and Recognition paradigms? |
| 4.3.6 | Purified CTT Effects (Paradigms) | If we compute CTT (Classical Test Theory) scores using only IRT-retained items (post-purification), do conclusions differ from full-item CTT for paradigm-specific forgetting trajectories (Free Recall, Cued Recall, Recognition)? |
| 4.3.7 | Paradigm-Specific Variance Decomposition | What proportion of variance in forgetting rate is between-person versus within-person for each retrieval paradigm (Free Recall, Cued Recall, Recognition)? |
| 4.3.8 | Paradigm-Based Clustering | Can participants be grouped into latent classes based on paradigm-specific forgetting trajectories (intercepts and slopes for Free Recall, Cued Recall, and Recognition)? |
| 4.3.9 | Paradigm × Item Difficulty Interaction | Do easier items show faster forgetting than harder items, and does this differ by retrieval paradigm (Free Recall, Cued Recall, Recognition)? |
| 4.4.1 | Do Congruent and Incongruent Items Show Different Forgetting Rates? | Does schema congruence (common, congruent, incongruent) affect the trajectory of episodic forgetting over 6 days? |
| 4.4.2 | Congruent Items and Early Consolidation | Is the schema congruence effect on forgetting driven by differential consolidation (Day 0→1) or later decay (Day 1→6)? |
| 4.4.3 | Age × Schema Interactions | Does the effect of age on forgetting rate vary by schema congruence (common, congruent, incongruent)? |
| 4.4.4 | IRT-CTT Convergence | Do IRT theta scores and CTT mean scores yield the same conclusions about congruence-specific forgetting trajectories? |
| 4.4.5 | Purified CTT Effects | If we compute CTT scores using only IRT-retained items (post-purification), do conclusions differ from full-item CTT for congruence? |
| 4.4.6 | Schema-Specific Variance Decomposition | What proportion of variance in forgetting rate is between-person vs within-person for each congruence level (Common, Congruent, Incongruent)? |
| 4.4.7 | Schema-Based Clustering | Can participants be grouped into latent classes based on congruence-specific forgetting trajectories (intercepts and slopes for Common, Congruent, and Incongruent items)? |
| 4.5.1 | Source-Destination Spatial Memory Trajectories | Do pick-up locations (source: -U-) and put-down locations (destination: -D-) show different forgetting trajectories in VR episodic spatial memory? |
| 4.5.2 | Source-Destination Consolidation (Two-Phase) | Do source (-U-) and destination (-D-) memories show different consolidation patterns across the Early (Day 0→1) and Late (Day 1→6) retention periods? |
| 4.5.3 | Age Effects on Source-Destination Memory | Does age moderate the source (-U-) vs destination (-D-) memory difference, or the forgetting rate for either location type? |
| 4.5.4 | IRT-CTT Convergence for Source-Destination Memory | Do IRT theta scores and CTT sum scores show high convergence for source (pick-up location: -U-) and destination (put-down location: -D-) memory, validating RQ 5.5.1 findings are not measurement artifacts? |
| 4.5.5 | Purified CTT Effects for Source-Destination Memory | Does IRT-based item purification improve CTT-IRT correlation for source (-U-) and destination (-D-) scores, and does the purification-trajectory paradox replicate? |
| 4.5.6 | Source-Destination Variance Decomposition | What proportion of variance in source (-U-) and destination (-D-) memory is attributable to stable between-person differences (intercepts vs slopes)? |
| 4.5.7 | Source-Destination Clustering | Can participants be grouped into latent classes based on source (pick-up location: -U-) and destination (put-down location: -D-) memory patterns (intercepts and slopes)? |
