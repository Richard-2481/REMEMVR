# Chapter 6: Individual Differences

| RQ | Title | Research Question |
|-----|-------|-------------------|
| 6.1.1 | Do cognitive tests predict overall REMEMVR ability? | Do established neuropsychological tests (RAVLT, BVMT, NART, RPM) predict overall episodic memory ability as measured by REMEMVR theta scores? |
| 6.1.2 | Intercept vs Slope Prediction | Do cognitive tests predict baseline ability (Day 0 intercept) more than forgetting rate (slope), consistent with tests measuring encoding but not consolidation? |
| 6.1.3 | Domain-Specific Prediction Patterns | Do verbal tests (RAVLT) preferentially predict What memory, visuospatial tests (BVMT) predict Where memory, and neither predicts When memory? |
| 6.1.4 | Unique REMEMVR variance unexplained by all predictors? | What proportion of REMEMVR variance remains unexplained after accounting for ALL available predictors (cognitive tests, demographics, self-report)? |
| 6.2.1 | Age Moderation of Test-VR Relationship | Does age explain variance in REMEMVR performance beyond what cognitive tests predict? If not, VR may compensate for age-related decline. |
| 6.2.2 | Do cognitive tests attenuate age effects on REMEMVR? | What proportion of age-related variance is attenuated when controlling for cognitive tests? Complete attenuation suggests tests capture all age-sensitive processes; partial attenuation suggests REMEMVR captures additional age-sensitive processes. |
| 6.2.3 | Age x Cognitive Test Interaction | Do cognitive tests predict REMEMVR differently for younger vs older adults? Tests may be better predictors in older adults if they tap compensatory processes. |
| 6.2.4 | VR Scaffolding Validation | Does REMEMVR show age-invariance while RAVLT shows age decline in the same sample? This formally tests the VR scaffolding hypothesis. |
| 6.3.1 | Do cognitive tests predict confidence trajectories? | Do cognitive tests predict confidence ratings (IRT-scaled) as they predict accuracy? This tests whether metacognition shares predictors with memory. |
| 6.3.2 | Cognitive Predictors of Calibration Quality | Do cognitive tests predict who is well-calibrated (confidence matches accuracy) vs overconfident (confidence exceeds accuracy)? |
| 6.3.3 | Cognitive Predictors of High-Confidence Errors | Do cognitive tests predict who makes more high-confidence errors (HCE)? Ch6 found 15-20% stable HCE rate - do individual differences have cognitive predictors? |
| 6.3.4 | Does DASS predict metacognition more than memory? | Does anxiety/depression predict metacognitive accuracy (confidence, calibration) more than memory accuracy? |
| 6.3.5 | Does confidence-accuracy gap predict cognitive reserve? | Do individuals with high confidence AND high accuracy (well-calibrated high performers) show signs of cognitive reserve? |
| 6.4.1 | Does RAVLT Free Recall predict REMEMVR Free Recall > Recognition? | Does RAVLT (a verbal free recall task) show stronger prediction for REMEMVR Free Recall than Recognition, consistent with process-specific transfer? |
| 6.4.2 | BVMT predicts Where more than What | Does BVMT (visuospatial memory test) show stronger prediction for Where (spatial location) than What (object identity)? |
| 6.4.3 | RPM Predicts Temporal Integration Performance | Does RPM (fluid intelligence) predict performance on items requiring integration of What+Where+When information? |
| 6.5.1 | Self-Report Predictors of REMEMVR Performance | Do self-reported factors (typical sleep, education level, VR experience) predict REMEMVR performance? |
| 6.5.2 | Does DASS predict memory performance? | Do depression, anxiety, or stress (DASS-21 subscales) predict REMEMVR episodic memory accuracy? |
| 6.5.3 | Memory Strategies Predicting Performance | Do self-reported memory strategies (rehearsal, visualization, mnemonics) predict REMEMVR performance? |
| 6.5.4 | Per-Test Sleep Effects on Same-Test Performance | Does sleep quality BEFORE each test predict THAT test's performance, demonstrating within-person state-dependent sleep effects? |
| 6.6.1 | Cognitive Tests Predicting Individual Differences in Forgetting Rate | Do cognitive tests predict the rate of forgetting (slope), or only initial encoding (intercept)? |
| 6.6.2 | Does RAVLT Delayed predict REMEMVR slope? | Does RAVLT forgetting (T5 - Delayed Recall) predict REMEMVR forgetting rate? |
| 6.6.3 | ICC slope replication across domains | Does the ICC_slope pattern (21% between-person variance) replicate across What, Where, When domains? |
| 6.7.1 | Reverse Inference - Can REMEMVR predict RAVLT? | Can REMEMVR performance predict standard test performance (RAVLT, BVMT)? If REMEMVR is a "purer" episodic measure, it should predict traditional tests. |
| 6.7.2 | Discrepancy Analysis - Who diverges? | Who shows RAVLT-REMEMVR divergence (high on one, low on other)? What characterizes these individuals? |
| 6.7.3 | Alternative RAVLT Scoring | Does RAVLT Learning Slope (T5-T1/T1) predict REMEMVR better than RAVLT Total? Can we suggest better RAVLT interpretation? |
| 6.7.4 | Clinical Profiles - False Negatives | Can we identify "false negatives" - individuals with low RAVLT but normal REMEMVR? These may have intact ecological memory despite poor lab performance. |
| 6.8.1 | Distinct REMEMVR memory profiles? | Are there distinct latent profiles of REMEMVR performance (e.g., "generalists" vs "specialists" vs "low performers")? |
| 6.8.3 | Parsimonious Predictive Model with Cross-Validation | What is the most parsimonious model to predict REMEMVR episodic memory performance, and how well does it generalize to unseen data? |
| 6.8.4 | Multivariate vs univariate prediction | Does predicting all three domains jointly (multivariate) outperform separate domain predictions? |
| 6.9.1 | Supplementary Validation Analyses for Individual Differences | Do the key individual differences findings from Chapter 6 survive formal statistical testing, domain extension, temporal isolation, and range-restriction correction? |
