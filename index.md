# REMEMVR Supplementary Materials

This repository contains the supplementary materials for the doctoral thesis:

> **Measuring Episodic Memory with Virtual Reality**

---

## Appendix Contents

| Appendix | Description |
|----------|-------------|
| [A1. IRT Item Parameters](docs/A1_irt_parameters) | Discrimination, difficulty, domain, retrieval condition, and purification status |
| [A2. Model Comparisons](docs/A2_model_comparisons) | AIC, ΔAIC, and Akaike weights for the forgetting function set |
| [A3. Participant Information Sheet](docs/A3_participant_information) | Information provided to participants prior to enrolment |
| [A4. Consent Form](docs/A4_consent_form) | Informed consent document signed by all participants |
| [A5. Ethics](docs/A5_ethics) | Ethics application (2022/HE001298) and approval letter |
| [A6. Scoring Matrix](docs/A6_scoring_matrix) | Score sheet used during cognitive testing sessions |
| [A7. Encoding Scripts](docs/A7_encoding_scripts) | Experimenter scripts for all four VR rooms |
| [A8. Scoring Rules and Foil Materials](docs/A8_scoring_rules) | Test questions, scoring criteria, and recognition foils |
| [A9. Analysis Code](docs/A9_analysis_code) | Python analysis scripts organised by thesis chapter |
| [A10. VR Application](docs/A10_vr_source) | Unity source files for the REMEMVR virtual environment |
